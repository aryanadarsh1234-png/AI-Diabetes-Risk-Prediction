# 🩺 AI-Based Diabetes Risk Prediction System

A Machine Learning web application that predicts diabetes risk using health indicators.

## Features

- Diabetes Risk Prediction
- Random Forest Classifier
- Streamlit Web Interface
- Data Preprocessing
- Risk Percentage Gauge
- Model Performance Metrics

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- Matplotlib

---

## Dataset

CDC BRFSS 2015 Diabetes Health Indicators Dataset

250,000+ Records

21 Features

---


## Installation

```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
```
